# SilkCircuit Slack Theme

## Quick Import (Recommended)

1. Open Slack → Preferences → Themes → Import
2. Copy and paste one of these theme strings:

### Neon (Default)
```
#2E1B7A,#E135FF,#50FA7B,#FF79C6
```

### Vibrant
```
#1A0033,#FF00FF,#00FF66,#FF00CC
```

### Soft
```
#3E3456,#E892FF,#66FF99,#FF99FF
```

### Cyan Accent
```
#0A0816,#80FFEA,#50FA7B,#FF79C6
```

## Manual Application

If you prefer to set colors individually:

**Neon Theme (Default)**
- **System navigation:** #2E1B7A
- **Selected items:** #E135FF
- **Presence indication:** #50FA7B
- **Notifications:** #FF79C6

## Tips

- Enable "Window gradient" for a smooth blend between navigation and selected items
- The purple navigation provides a nice base
- Pink notifications ensure you never miss a mention
- Green presence keeps the standard "online" association
